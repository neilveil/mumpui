MumpUI / Typography

MumpUI comes prepacked with typography styling, making it ideal for rendering content, blogs, stories and more. Typography elements are of two types:

  • Inline elements
    • Bold <b></b>
    • Italics <i></i>
    • Underline <u></u>
    • Highlight <mark></mark>
    • Strike <del></del>
    • Code <code></code>
    • Link <a></a>
  • Block elements
    • Headers h1 h2 h3 h4
    • List
      • Ordered list <ol></ol>
      • Unordered list <ul></ul>
    • Table <table></table>
    • Divider <hr />
    • Blockquote <blockquote></blockquote>
